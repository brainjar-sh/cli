# @brainjar/cli (deprecated)

> **This package is deprecated.** brainjar 1.0 ships as a single Go
> binary at
> [brainjar-sh/brainjar](https://github.com/brainjar-sh/brainjar).
> Install it with:
>
> ```sh
> curl -fsSL https://get.brainjar.sh/brainjar/install.sh | sh
> ```
>
> The Go binary supersedes this package and the standalone server.
> See [DEPRECATED.md](DEPRECATED.md) for the rationale and migration
> notes. The last release on npm is `0.7.6`.

The original README is preserved in git history at any tag prior to
`v0.7.6`.
